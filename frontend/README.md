# 💻 前端展示层 (Frontend Layer)

的交互门户，负责展示质押状态、习惯日志及惩罚记录。

### 核心功能描述：
- **质押控制面板 (Staking UI)**：用户连接钱包并向 `HabitEscrow` 合约发送质押资金的交互界面。
- **自律状态看板 (Dashboard)**：
    - 展示当前质押金额。
    - **习惯日志**：实时滚动展示 AI 的裁判结论（Verdict）和对应的交易哈希（Transaction Hash）。
- **反馈系统**：当质押被没收时，展示 AI 的嘲讽通知。
